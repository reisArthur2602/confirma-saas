Design System

Catálogo vivo de tokens e componentes usados nos protótipos do Confirma — dashboard, documentação, blog, landing, login e onboarding. IBM Plex Sans para texto, IBM Plex Mono para dados técnicos, sem framework CSS: tudo construído com estilo inline direto nos protótipos.

Animações

Três keyframes cobrem todo o sistema — sem exceção. Passe o mouse pelos exemplos.

cfmFade
Entrada de telas, dropdowns, mensagens
cfmDrop
Abertura de dialogs, resultados
cfmPulse
Dots "ao vivo" — status, ambiente, teste em andamento
Fundamentos

Ícones

SVG inline, fill="none", stroke contextual. stroke-width 1.8–2.6 (maioria 1.9–2.4), tamanhos 13–34px, viewBox="0 0 24 24", cantos arredondados. Estilo Lucide/Feather.
